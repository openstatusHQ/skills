# Openstatus Skills for Agents

A collection of AI agent skills focused on monitoring tasks.

## Installation

### Option 1

Install it via [npx skills](https://github.com/vercel-labs/skills):

```bash
# Install all skills
npx skills add openstatushq/skills

# Install specific skills
npx skills add openstatus/skills --skill global-speed-checker

# List available skills
npx skills add coreyhaines31/marketingskills --list
```

### Option 2

Install it via Claude Code plugin:

```
# Add the marketplace
/plugin marketplace add openstatushq/skills

# Install all skills
/plugin install openstatus-skills
```
